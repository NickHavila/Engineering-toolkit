# Engineering Toolkit

A console application written in C that performs basic engineering calculations.

## Features

- Average Velocity
- Average Acceleration
- Force Calculation
- Kinetic Energy
- Projectile Motion *(Coming Soon)*

## Technologies

- C
- GCC
- Standard C Library
- math.h

## Project Structure

```
Engineering-Toolkit/
| main.c
| physics.c
| physics.h
| README.md
```

## How to Compile

Using GCC:

```bash
gcc main.c physics.c -o toolkit -lm
```

## How to Run

Windows:

```bash
toolkit.exe
```

Linux/macOS:

```bash
./toolkit
```

## Example

```
===============================
Engineering Toolkit
===============================

1 - Physics
0 - Exit

Select an option:
```
